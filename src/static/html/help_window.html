This tool aims to predict if a mass pathology is malignant or non malignant. In order to generate the predictions, you should have to introduce an excel file with the following fields:

  • ID: Identification field for every region of interest. This field must be unique.
  • FILE_PATH: Absolute file path of the image. Only dicom and pgm formats are available.
  • X_CORD: X-axis coordinates for center of abnormality. Coordinate system origin is the bottom-left corner.
  • Y_CORD: Y-axis coordinates for center of abnormality. Coordinate system origin is the bottom-left corner.
  • RAD: Approximate radious (in pixels) of a circle enclosing the abnormality. Every radious lower than 15 pixels will be ommited.

Once the input excel has been selected, you must specify one folder in order to store application outputs. Generated outputs are:

  • Excel File: File with benign-malign mass classification for every observation.
  • Malignant File: Folder with malignant clasificated mass images in png format. Every image has its regions of interest marked in red.
  • Benign File: Folder with benign clasificated mass images in png format. Every image has its regions of interest marked in red.
  • Error log files: Text files generated during execution informing about any issue.
